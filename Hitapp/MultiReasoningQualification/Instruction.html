What is this HIT about?

We are trying to build a question-answering system which is able to retrieve the answer by reasoning over multiple facts. This is the way we human typically use to find an answer. For example, when this question is asked:

Has there always been a rivalry between Daffy Duck and Porky Pig?

We might find the answer by

  1. Searching Wikipedia for Daffy Duck.
  2. "Daffy first appeared in Porky's Duck Hunt, released on April 17, 1937."
  3. "Porky shoots the whiskey-filled barrel but Daffy escapes" or "Daffy bites Porky on the nose"
  4. Conclude, Yes, Daffy and Porky have been rivals since their first appearance on screen.

However, for now, if you search a question in a search engine like Google or Bing, you are mostly like to get the answer from only one snippet of text. They don’t have the ability to answer the above question because, as you can see, the answer is concluded by reasoning the facts from multiple resources. We believe that this reasoning ability can greatly improve the question-answering ability of search engines, so we create this Hitapp and asking help from you. Appreciate!

The Format:

In each hit, you will see a group of QnA pairs with supporting facts. Your duty is to determin if a the QnA pair is good enough and can be supported by ALL showing facts. You need to answer the given question for each QnA pair to describe the quality.

A good QnA pair should be:

  1. Shouldn’t be answered by only one fact:

    Facts:

    1. Hawa was born on December 16th, 1937 in Umarwas.
    2. Hawa enrolled in the Indian Army in 1956, and became the champion of the Western Command in 1960 by defeating the defending champion, Mohabbat Singh.

    Bad question:When is Hawa’s birthday.(can be answer by the first fact)

    Good question: How old was Hawa when he enrolled in the Indian Army? (need both facts)

  2. Has definitive answer: The answer to your query cannot be too subjective or change from time to time.
    • How good is the view of Chicage from Willis Tower skydeck? (Too subjective)
    • What is the weather in Manhattan? (Time sensitive)
  3. Not about simple facts: The query cannot be answered blindly based on common sense (i.e. without using any of the text snippets).
    • Can a boat float on the water?
    • How many wheels does a car have?
