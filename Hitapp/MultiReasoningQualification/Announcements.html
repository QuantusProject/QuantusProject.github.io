In this HitApp, you will be asked to judge the quality of some multi-reasoning Question-and-Answer(QnA) pairs.

A multi-reasoning QnA pair is an Question-and-Answer pair which can only reasoning from several different facts/text snippets.

This task is used as a qualification task. Once you passed this qualification task, you will have the access to more hits created by us!

Here are some explaination of each question:

  1. Can you understand what is being asked without reading the facts?: If the question is grammatically correct, please check "Yes".
  2. Is this answer really a valid and correct answer?: If the question is grammatically correct and can answer the question, please check "Yes". You don't need care about the facts in this question.
  3. Can this answer be inferred from the facts?: If all the information used in the answer can be found in all or some of the facts, check "Yes".
  4. Are all facts necessary to answer the question?: If all the facts are needed to answer the question, check "Yes".

More detailed information are coming, we are looking forward to see the result from your work!
