1. Must all facts be used to answer questions?

    Yes, the answer should be derived from all facts. If any of the fact is not necessary to answer the question, please mark it as bad and mark it in the text box.

  2. Does the facts need to have strong relationship between each other?

    Yes, the question should not make up relationship between the facts. Instead, the facts themselves should be related, either in the same domain or descripbing similar/same entity/topic. For example: "wood" and "tree" are some related entity. However, "Panda Express" and "panda" are not, because one is the name of a restaurant, but the other one is a kind of animal.   3. Do I need to try to understand the question, answer and facts before making decision?

    Yes! You can try to use any search engine to understand each QnA pair and then make the decision. This is very important to give high quality result.

  4. What should I do if it contains duplicate facts?

    You can try to remove the duplicate facts. After removing any one of them, we still have more than 2 facts left to answer the question, you can mark it as good. However, if only 1 fact left, please mark it as bad. The dupication in meaning is also considered as duplication.
