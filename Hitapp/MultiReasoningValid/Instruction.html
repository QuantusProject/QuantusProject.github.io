What is this HIT about?

We are trying to build a question-answering system which is able to retrieve the answer by reasoning over multiple facts. This is the way we human typically use to find an answer. For example, when this question is asked:

Has there always been a rivalry between Daffy Duck and Porky Pig?

We might find the answer by

  1. Searching Wikipedia for Daffy Duck.
  2. "Daffy first appeared in Porky's Duck Hunt, released on April 17, 1937."
  3. "Porky shoots the whiskey-filled barrel but Daffy escapes" or "Daffy bites Porky on the nose"
  4. Conclude, Yes, Daffy and Porky have been rivals since their first appearance on screen.

However, for now, if you search a question in a search engine like Google or Bing, you are mostly like to get the answer from only one snippet of text. They don’t have the ability to answer the above question because, as you can see, the answer is concluded by reasoning the facts from multiple resources. We believe that this reasoning ability can greatly improve the question-answering ability of search engines, so we create this Hitapp and asking help from you. Appreciate!

The Format:

In each hit, you will see a group of QnA pairs with supporting facts. Your task is to determine if shown QnA pair is a good reasoning based QnA and needing EACH of the facts shown for distinct information to get the answer.

A good QnA pair should be:

  1. Shouldn’t be answered by only one fact:

    Facts:

    1. Hawa was born on December 16th, 1937 in Umarwas.
    2. Hawa enrolled in the Indian Army in 1956, and became the champion of the Western Command in 1960 by defeating the defending champion, Mohabbat Singh.

    Bad question:When is Hawa’s birthday.(can be answer by the first fact)

    Good question: How old was Hawa when he enrolled in the Indian Army? (need both facts)

  2. The Question and Answer should be grammatically correct and has no speeling error;

    Bad question: Was George a techer when he was 21 years old?("Teacher" is spelled wrong)

  3. The Question should not be too easy or can be answered by common sense.

    Bad question: How many wheels does a car have?

    Explanation: Since everybody knows a car have 4 wheels in most cases, this question does not need to use any facts to answer.

  4. The Question and Answer themselves should be correct.

    Bad question: Which has stronger smell, rose or flower?

    Explanation: rose is a type of flower, so they are not comparable

  5. The facts used to answer the question should be reasonably related.

    Facts:

    1. Keanu Reeves was apparently such an outstanding hockey goalie that his college teammates called him "The Wall."
    2. American actor Timothee Chalamet has been dubbed “Sweet Tea” by his fans in China.

    Bad question: Who's real last name is shorter? "The Wall" or "Sweet Tea"?

    Explanation: The two provided facts are not talking about same topic, does not have same entity. So basically, they are not related. We should avoid such question.

  6. Has definitive answer: The answer to your query cannot be too subjective or change from time to time.

    Bad question: How good is the view of Chicage from Willis Tower skydeck? (Too subjective)

    Bad question: What is the weather in Manhattan? (Time sensitive)

Work flow:

  1. Read the given question
  2. Answer the "*Tell us about this Question*" section,Select any tag available to mark up the Question, feel free to add others if you think it's need to describe the question
  3. Read the facts
  4. Answer the "Tell us about the facts:" section, use One word to describe the topic of the facts. Some candidates are given.
  5. Read the answer
  6. Answer the "*Tell us about this Answer*" section
